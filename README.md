# README

Readme generator. Features customizable templates that are fetched on load.

- [Template Directory](./public/templates)

## Uses

- [asdf](https://asdf-vm.com)

## Developing

```bash
asdf plugin add nodejs
asdf install

# Install dependencies
npm install

# Start dev server
npm run dev

# Testing
npm run test
```
